# Ola Ride Performance Analysis
Tools Used: SQL | Power BI

## Project Overview
This project analyzes Ola ride booking data to evaluate booking performance, cancellations, payment methods, vehicle usage, and ratings.

## Key Insights
- Cash and UPI are dominant payment methods
- Cancellations are a major operational issue
- Vehicle types serve different ride-distance patterns
- Ratings are consistent across vehicle types

## Business Value
The dashboard supports operational monitoring and helps identify areas to improve ride completion and efficiency.
